Detailed Description of Module

Webcam Image By ACS

Will add button to set image on user/partner form and on click of that button it will open window to take image using connected web-cam.

On Webcam Window you will see live video and on perfect time you can click on snap Photo. (you can click multiple time on snap photo till you get good snap.)

Once you get good photo click on Save image and it will update image of related record and open that record back.

NOTE: back redirection may take to the sepcific action only. Module will work with HTTPS domain only becuase of chrome (broser) Security. On local it will work without HTTPS.

This module is Generic so it can be extended further in custom modules also.
