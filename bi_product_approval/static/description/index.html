Product Approval in Sale, Purchase and Manufacturings

With the Product Approval Odoo App, users can create products that require approval. The product will be create default in draft stage then only product manager have access to approve the product with single click. Once approved product by product manager then the product will be available to create order like sale order, purchase order, manufacturing order etc.

Features

Product Manager Access Rights

User have to enable product manager option to given access of product manager.

Create Product in Draft Stage

By default Product will be created in draft stage.

Approve Product By Manager

Product manager have access to approve the product by single click.

Update Information in Draft Stage of Product

User can update the product information till in draft stage, Once product in approve stage, User will not be able to update information.
Demo Request Video Tutorial

Product Manager Access Rights

Under user configuration, User have to enable "Product Manager" option to review and approve the product.

Create Product by User

When user create any new product it should be set in draft stage.

When products in draft stage, User should not access those product until approval in any of the model like sale, purchase, MRP etc.

Approve Product By Manager

Manager can approve product by clicking on "APPROVE" button on product.

Approved Product By Manager

Once clicking on "APPROVE" button, Product should be moved in "APPROVE" stage and also user or manager should not be able to update the product information.

Once products approved by manager, User should access those products in any of the model like sale, purchase, MRP etc.

Reset To Draft Product By Manager

Manager can reset product into draft stage by clicking on "RESET TO DRAFT" stage.

Once product in draft stage, User or manager should be able to update the product information.
