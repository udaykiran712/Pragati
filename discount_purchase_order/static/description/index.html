Odoo Discount On Purchase Order

Discount on Purchase Order module adds the fixed type discount to Odoo, which is unavailable by default. The Odoo app can add fixed or percentage-type purchase Orderline discounts. You can now add Global Discount Category that can be fixed or a percentage type based on selection.

Why Do You Need Odoo Discount on Purchase Orders?

By default, Odoo does not allow discounts on purchase orders. However, you must have flexibility in giving discounts in Odoo while creating purchase orders.

This Odoo app adds a fixed discount type in Odoo which is not available by default. Also, it permits managing discount types for each purchase order. The Odoo admin can add Orderline or global discount on the purchase order.

The module allows you to provide multiple options to apply discounts in Odoo on purchase orders, invoices, and bills (after installing the base module, you can add discounts on invoices and bills).

Adding Discount Types on Purchase Orders

By default, there is no discount field in the Purchase orders. Also, you cannot add a fixed discount in Odoo.

However, with our module, you can add a fixed discount on purchase orders. Moreover, it allows you to add global discounts on the orderline. You can apply a discount on taxed included and excluded amounts of the product.

Added Global Discount Category for the Entire Purchase Order

By default, you can not add any global discount on purchase orders in Odoo.

With this module, you can add the global discount applied to the total purchase order. Also, the module permits adding fixed or percentage discount types on the whole purchase order. Again you can give taxed or untaxed purchase order amounts as set in the settings.
