POS Stock Management Odoo Apps

This Odoo apps is used to display stock in POS interface also the seller can select and display different type of stock like "Qty on hand" and "Qty Available" in POS and Allow/Deny POS sale Order when product is out of stock. You can display stock of All & Specific stock location in POS. You can configure stock position as "Top Right", "Top Left" and "Bottom Right" on POS screen with customize stock background color and font color. You can view the low stock of products in list view on POS screen. When you added product into cart, the total number of items will be calculated and show in POS receipt.Also Stock will be automatically sync when order get placed in Odoo POS inventory management App.

POS Stock Configuration

Under Point of Sale > Configuration > Point of sale menu user need to enable "Display Stock in POS" option to see stock for product on point of sale screen.

User can also select "Stock Type" option as "Qty on Hand, Qty Available" and as selected product stock will displayed on pos screen.

User can also configure "Stock Position" as "Top Right, Top Left and Bottom Right" on the pos screen.

User can also customize background color and font color of the stock.

User can also select "Show Stock of" option as "All Warehouse" or "Current Session Warehouse", if all warehouse option selected then stock from all warehouse will displayed, if current session warehouse option selected then user need to select "Stock Location" and stock from selected location only displayed.

Qty On Hand

User see total on hand quantity of product.

User see total on hand quantity of product from all warehouse is displayed on point of sale screen, if product is out of stock it will be red.

When user added products into cart, Total number of items should be calculated.

User see total number of items in POS receipt.

Qty Available

User need to select "Qty Available" option.

User see available quantity of all product from all warehouse is displayed on point of sale screen with selected stock position and color.

Product Low Stock

User have to enter the number to count the product as low stock.

User can click on 'Low Stock Product List' option to check the low stock of products in list view.

User can view the low stock of products with quantity in list view.

Auto Update Product Quantity

Now user no need to reload the POS screen every time after placing an order, Stock will be automatically updated on POS product screen.

Deny POS Order

Stock Quantity Validation

If user add pos product quantity more than available quantity in order line for a particular product then stock qty validation will be added even user selected "All Warehouse" or "Current Session Warehouse" option as configuration.

User can see warning as below image.

Stock quantity validation will not be applied for "Consumable" or "Service" type product. Although user can add more qty in the order line, it will go to the payment screen without validation.
