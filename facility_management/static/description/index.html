Application to manage Facility in Odoo.


Facility management contain many things but if we think about small company they need to manage cleaning process, repair or maintenance process on regular intervals for almost all assets.
In odoo now you can manage Facility registrar for following processing this module.
1> Office Cleaning Register
2> TV, AC of Car maintenance Register
3> Data Backup Process
And many more possibility based on requirement and thinking.

Create Your Facility Masters by entering proepr data and execution type. Based on selected execution intervals it will create Activity automatically.
Facility master tree view with color based on states.
Facility Activity form view. It can be created automatically and manually also with unique sequence number.
Facility Activity tree view with color based on states.
