WhatsApp Notification


This Module Adds functionality to configure WhatsApp API and send whatsapp messages using that api. By doing further customization messages can be sent from any odoo applications.


NOTE: Whatsapp Official Cloud API is used, So to use module user will need to have https://developers.facebook.com/docs/whatsapp/cloud-api Account and related configuration.

Detailed Description of Module

WhatsApp Integration By ACS

Configure your CHAT-API Credentials on company.

Send Message directly from contact & Check history for sent messages.

Announcement can be done to list of employees or Customers easily.

Send Message directly from appointment/patient & Check history for sent messages.

WhatsApp User and Manager Access Roles are there to manage it Properly.

NOTE: Option to read Conversation with customer. (For Last 100 Messages). (BETA)
